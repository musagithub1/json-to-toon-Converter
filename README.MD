## ğŸ§° How to Run on Another System (Friend's Computer)

1. **Install Docker.**
2. **Create a folder with this file inside:** `docker-compose.yml`:

```yaml
services:
  backend:
    image: musagithub1/json-to-toon-converter-backend:latest
    ports:
      - "8000:8000"

  frontend:
    image: musagithub1/json-to-toon-converter-frontend:latest
    ports:
      - "8080:80"
    depends_on:
      - backend
```

3. **Run:**

```bash
docker compose pull
docker compose up -d
```

4. **Visit:**
   - Frontend â†’ `http://localhost:8080`
   - API Docs â†’ `http://localhost:8000/docs`

**Stop anytime:**

```bash

```
